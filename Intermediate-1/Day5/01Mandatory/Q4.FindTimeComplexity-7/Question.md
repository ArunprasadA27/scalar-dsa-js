## Q4. Find Time Complexity - 7 - Objective

### What is the time complexity of the following code :

```
for (int i = 0; i < n; i++) {
    for (int j = 0; j <= i; j++) {
        System.out.print(i + j + " ");
    }
    System.out.println();
}
```

### Answer
O(nlogn)
